HOW TO ADD YOUR OWN IMAGES
===========================

No editing needed. Just save each photo using the EXACT filename listed
below (lowercase, matching exactly), and place it inside this folder:

    details/images/

Once a file with the right name is in this folder, refresh the matching
page in your browser and the photo will replace the placeholder
automatically. If a file is missing, the page just keeps showing the
placeholder box -- nothing breaks.

Recommended: JPG, under ~2MB, roughly landscape or 4:3.
